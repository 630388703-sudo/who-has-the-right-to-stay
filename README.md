# WHO HAS THE RIGHT TO STAY?

**沉浸式数字展览 / Immersive Digital Exhibition**

一个关于校园空间权利的网页沉浸式体验。以校园保洁员的第一视角，探讨公共空间是否真正对所有人平等开放。

> "劳动者虽然长期存在于公共空间中，却缺少真正属于自己的停留空间。"

---

## 体验流程

| 阶段 | 时间 | 空间 |
|------|------|------|
| 01 启动系统 | 06:30 | — |
| 02 进入工具间 | 06:30 | 后勤工具间（唯一可停留空间） |
| 03 开始巡查 | 07:15 | 教学楼走廊 |
| 04 身份对比 | 09:40 | 同一空间，两种权限 |
| 05 休息被打断 | 12:40 | 楼梯角落 |
| 06 结束工作 | 18:00 | — |

---

## 本地运行

直接用浏览器打开 `index.html` 即可，无需任何构建工具或依赖。

```bash
# 或者用 Python 起一个本地服务器
python3 -m http.server 8080
# 访问 http://localhost:8080
```

快捷键：按 `→` 或 `空格` 推进场景。

## 项目背景

本项目由数字媒体艺术专业学生创作，作为「Vibe Coding + 沉浸式数字展览」课程项目。

核心议题：**公共空间是否真正对所有人平等开放？**

视觉风格：Documentary + System UI，极简黑白 + 灰绿系统UI，营造监控感与后勤空间的压迫感。
